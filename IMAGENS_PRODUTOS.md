
# Guia de Imagens dos Produtos

## Como adicionar as imagens reais:

1. Baixe as imagens dos produtos dos fabricantes
2. Renomeie-as conforme os nomes abaixo
3. Coloque na pasta `static/images/products/`

## URLs Temporárias (Placeholder):

Enquanto não tiver as imagens, o sistema usa placeholders automáticos.
Para melhores resultados, use imagens reais com:
- Dimensão: 800x600px (ou proporção 4:3)
- Formato: JPG ou PNG
- Tamanho: máximo 500KB por imagem
- Fundo branco ou transparente

## Fontes de Imagens Gratuitas (Alternativa):

Se não conseguir imagens dos fabricantes, pode usar:
- Unsplash: https://unsplash.com/s/photos/solar-panel
- Pexels: https://www.pexels.com/search/solar%20panel/
- Pixabay: https://pixabay.com/images/search/solar%20panel/

## Script para baixar imagens de exemplo:

Execute este comando para criar imagens placeholder temporárias:
```bash
mkdir -p static/images/products
mkdir -p static/images/testimonials
mkdir -p static/images/projects
```
